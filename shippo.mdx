In this article, you'll learn how to integrate Shippo with the Alloy platform. Once you've connected Shippo, you'll be able to sync data from Shopify, BigCommerce, Postscript, Omnisend, and other apps to view customer data in one location.

### Example Use Cases

1. Connect Airtable to Shippo so that when order is approved in Airtable, shipping label will be auto-generated and paid for
2. Create shipping labels for gift items after customers have spent over $X amount on store

### Get Shippo credentials

1. Login to Shippo
2. Go to Settings -> Integrations -> API -> Generate Token to get your API Token

### Connect to Alloy

1. In Alloy, create a new workflow
2. Select Shopify Trigger, or any app you'd like to trigger the workflow. Set up the app.
3. Choose Shippo as the second block in your workflow by clicking on the + icon next to the trigger
4. Choose any action, such as Create a Shipping Label
5. In the next step, paste your Shippo token into the field then press Validate
6. Proceed to the next step and fill in the required fields
