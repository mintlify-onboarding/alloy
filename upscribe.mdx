In this article, you'll learn how to set up your Upscribe account to work with the Alloy platform. Once you are done, you'll be able to sync data from one of our 200+ apps into Upscribe and vice versa.

### Add Your Upscribe Account to Alloy

Read the instructions below to get set up, or watch the following video to learn how Alloy's Upscribe integration works

**Setup Instructions**

1. In your dashboard, create a new workflow from scratch
2. Select the Upscribe trigger  
![](/images/7a4af32-image.png)
3. Enter your Shopify Store Domain and your Upscribe API Key. Your API Key can be found by going to your Shopify store > Apps > Upscribe > Settings > Advanced Settings > General Settings > Create an API Key.  
![](/images/ce4e7d6-image.png)
4. Choose an event to trigger the workflow. For example, if you choose Order Created, the workflow will run every time an order is created in your Upscribe store.  
![](/images/7b7814a-image.png)
5. You can now continue building the rest of your workflow! Select additional logic or action blocks to automate work across other apps.
