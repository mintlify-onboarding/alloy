In this article, you'll learn how to set up your Commerce.js account to work with the Alloy platform. After completion, you'll be ready to sync data into Commerce.js from one of our 200+ apps and vice versa.

### Example Use Cases

1. Get notified when a new order exceeds a limit
2. Notify customers when their order has been refunded
3. Enroll your customers in a newsletter upon account creation (Klaviyo)

### Add Your Commerce.js Account to Alloy

Read the instructions below to get set up, or watch the following video to learn how Alloy's Commerce.js integration works

**Setup Instructions**

1. In your dashboard, create a new workflow from scratch
2. Select the Commerce.js trigger  
![](/images/9e930dd-image.png)
3. Enter your API Key. To find your API Key, simply click on the “Get Your Credentials” button within the Commerce.js trigger.  
Once you are in your Commerce.js account, click on Add and then create your new API Key.  
![](/images/2823f3e-image.png)
4. Choose an event to trigger the workflow. For example, if you choose Order Created, the workflow will run every time an order is created in your Commerce.js store.  
![](/images/9c886b5-image.png)
5. You can now continue building the rest of your workflow! Select additional logic or action blocks to automate work across other apps.
