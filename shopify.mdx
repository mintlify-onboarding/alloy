### Add Your Shopify Account to Alloy

Read the instructions below to get set up, or watch the following video to learn how Alloy's Shopify integration works

**Setup Instructions**

1. In your dashboard, create a new workflow from scratch
2. Select the Shopify trigger  
![](/images/f0b5ded-image.png)
3. Enter your shop subdomain. Your shop subdomain can be found in the URL of your Shopify dashboard and consists of everything before .myshopify.com.  
![](/images/b90e6f5-Screenshot_2022-11-29_110807.png)![](/images/f0f85fe-image.png)
4. Choose an event to trigger the workflow. For example, if you choose Order Created, the workflow will run every time an order is created in your Shopify store  
![](/images/19f56ba-image.png)
5. You can now continue building the rest of your workflow! Select additional logic or action blocks to automate work across other apps.
