In this article, you'll learn how to integrate Omnisend with the Alloy platform. Once you've connected Omnisend, you'll be able to sync data from Shopify, BigCommerce, Postscript, ShipBob, and other apps to view customer data in one location.

### Example Use Cases

1. Send update messages to customers via email, SMS, or Facebook Messenger
2. Add tagged customers to remarketing lists for Google Ads
3. Automatically tag and segment customers based on specific criteria
4. Flag and canceling high-risk orders before capturing the payment

### How to setup the integration

Get Omnisend credentials

1. Login to Omnisend
2. Go to Dashboard -> User Profile -> Integrations -> API Keys to view the credentials

### Connect to Alloy

1. In Alloy, create a new workflow
2. Select Shopify Trigger or any app you'd like to trigger the workflow. Set up the app.
3. Choose Omnisend as the second block in your workflow by clicking on the + icon next to the trigger
4. Choose any action, such as **Create Contact**
